Chanjo: Clinical coverage done evolved
=======================================
Release v0.0.1-beta

Chanjo is a clinical sequencing coverage analysis tool. Gene, transcripts, and exons over obscure BED intervals. Move coverage analysis beyond "average coverage" by introducing additional metrics like completeness.

Welcome to Chanjo's documentation. This resource will help you with installation, provides a quickstart guide and catches you up on recent developments if you like ... More detailed information about the architechture and design of Chanjo is found ... This can be useful if you would like to contribute to the development on Chanjo.
